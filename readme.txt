install caution:
1.wix_tool is for windows.wix_tool_Lx is for Linux
2.for windows :
	(1)Download wix_tool\
	(2)enter the wix_tool\C_space\
	(3)execute orders: (1)mingw32-make -f wix_DLL.makefile
		             (2)mingw32-make -f wix_program.makefile
	(4)join the wix_tool to systempath
	(5)you can use it in cmd or powershell and get the help by input "wix --help"
3.for Linux:
	(1)Download wix_tool_Lx
	(2)enter the wix_tool_Lx/C_space/
	(3)execute orders: (1)make -f wix_DLL.makefile
		             (2)make -f wix_program.makefile
	(4)join the wix_tool_Lx to systempath
	(5)you can use it in terminal and get the help by input "wix --help"
4.thanks for installation!!
